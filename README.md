# Naya Bharat Engineering — Homepage

Modernized homepage for [Naya Bharat Engineering](http://nayabharatengineering.com) — a B2B + retail hub for mining, crushing, and heavy construction spares (Kabrai, Uttar Pradesh).

## Live site

[https://kaustubh10-01.github.io/nayabharat-website/](https://kaustubh10-01.github.io/nayabharat-website/)

## Stack

Single self-contained `index.html` — embedded CSS and JS, Google Fonts via CDN. No build step.

## Local preview

Just open `index.html` in any modern browser.

## Design source

Created in [Claude Design](https://claude.ai/design). See the design bundle's chat transcript for the design rationale (typography, color, spacing system).
